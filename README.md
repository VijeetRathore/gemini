# Branch Compliance Audit Form

## Overview
Automated multi-page audit form for branch compliance tracking.

## Features
- Dynamic language switching (Vernacular Support).
- Mandatory field validation with UI highlighting.
- Integrated with Google Sheets API.
- Camera access for mandatory photo uploads.

## Installation
1. Clone the repo: `git clone [your-repo-link]`
2. Open `index.html`.
3. Update `SCRIPT_URL` with your Apps Script Web App URL.
